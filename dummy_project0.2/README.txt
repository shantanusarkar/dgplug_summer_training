How to install myshell?
=====================

Link to download the tar.gz file : https://testpypi.python.org/packages/source/d/dummy_project/dummy_project-0.2.tar.gz#md5=a600edad589e24915236b071b3fc554b

1. Extract the tar.gz file
2. Open terminal > go to the extracted directory.
3. As root user type "python setup.py install" and press enter.
